Workflow WebApp

Money Transfer Demo

Golang sample web app with a mysql database table acting as a task queue feeding money transfer requests that are processed using Temporal Workflow and Activities.

There is a background thread that polls the local request table and submits a new Temporal Workflow when required, the resultant workflow and run ids are updated in the local table as well as the end return result of the workflow.

Local Account Managment

Golang CRUD app managing a mysql database table of user account entries with a balance column.
Accounts can be listed, created and deleted.

Note: At this time there is no association between these accounts and the Money Transfer demo account entries, these are currently hardcoded in the banking-client implementation program.
